## overview
* debug()     - 标记函数进行调试
* undebug()   - 取消标记函数进行调试
* browser()   - 允许单步通过函数的执行
* trace()     - 修改函数以允许暂时插入调试代码
* untrace()   - 取消追踪并删除临时代码
* traceback() - 打印导致了最后未捕获错误的函数调用序列

## browser
调试时,输入n或按回车键执行当前语句并移动到下一行

输入c继续执行到函数的末尾,没有单步执行

输入 where 显示调用的堆栈

输入 Q 停止执行并立即跳到顶层

其他的 R 命令诸如 ls() 、 print() 和赋值语句也可以在调试器中提交
